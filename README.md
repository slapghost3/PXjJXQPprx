# 新生入学管理系统 python702

## 项目概述

新生入学管理系统是基于 Python Django 框架开发的一款管理软件，旨在提高新生入学相关工作的效率，为管理员和学生提供便捷的服务。

## 技术栈

- **开发语言**：Python
- **开发框架**：Django
- **数据库**：MySQL
- **架构**：B/S

## 功能模块

### 系统角色

- **首页**：展示系统概览和最新信息。
- **个人中心**：用户管理个人信息。

#### 管理员功能

- 管理专业、班级、学生信息
- 管理商品信息
- 新生签到
- 管理交流论坛
- 系统设置
- 处理订单

#### 学生功能

- 查看个人信息
- 新生签到
- 我的收藏

## 运行环境

- Python 环境推荐版本：3.8+
- MySQL 数据库版本：5.7+

## 部署步骤

1. 配置 Python 环境及所需依赖
2. 安装并配置 MySQL 数据库
3. 导入数据库文件
4. 启动 Django 应用

## 目录结构

```
project/
│
├── apps/ # 应用目录
├── db/ # 数据库文件
├── media/ # 媒体文件
├── static/ # 静态文件
├── templates/ # 模板文件
├── manage.py # 管理脚本
└── requirements.txt # 依赖列表
```
